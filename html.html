Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

Tags and attributes are the basis of HTML. They work together but perform different functions.

Tags are used to mark up the start of an HTML element and they are usually enclosed in angle brackets (<tagname>). For example, the 'p' tags, opening p tag <p> and the closing p tag </p>. Most tags must be opened and closed in order to function.

Attributes contain additional pieces of information. Attributes take the form of an opening tag and additional info is placed inside. For example, in of an image tag, <img src="mydog.jpg">, 'img' is the main tag and 'src' is the attribute that modified the image tag to display the correct image and and provide the required output.

HTML documents always starts with the code <!DOCTYPE html>. This tell the browser that the code in this document is HTML 5. The document itself should always be enclosed within html tags, <html>...</html>.

An HTML document is of two parts, the head and the body. The head section is used to contain information related to the document such as the documents title and meta data. The body section is the visible part of the webpage. That is everything visible in the browser window.

This is the basic model of an HTML 5 webpage

<!DOCTYPE html>
<html>
<head>
<title>Webpage Title</title>
</head>
<body>
<p>Hello World</p>
</body>
</html>

Given below are a few commonly used html tags

Element Tag Description
Heading <h1>...</h1> OR <h2>...</h1> OR <h3>...</h1> OR <h4>...</h1> OR <h5>...</h1> OR <h6>...</h1> OR Headings are defined with the <h1> to <h6> tags where <h1> is the biggest and <h6> is the smallest.
Paragraphs <p>...</p> Used to add paragraphs in the webpage.
Comments <!-- ... --> Adds comments to the HTML source documents.
Links <a>...</a> Links the document to other HTML pages and the 'href' attribute specifies the destination addressof the link.
Images <img src="image_link"> Adds the image pointed by the 'src' attribute into the HTML document.
Table <table>
    <tr>
        <td>...</td>
        <td>...</td>
        ...
    </tr>
</table> 		Adds a table into the HTML document. Each table row is defined with the <tr> tag. A table header is defined with the <th> tag. By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.
Unordered List <ul>
    <li>...</li>
    <li>...</li>
    ...
</ul>
 A list of items that will be marked with bullets (small black circles) by default.
Ordered List <ol type="1">
    <li>...</li>
    <li>...</li>
    ...
</ol>
 A list of items that will be marked with numbers by default. If type is 1 items will be numbered with numbers. Type 'A' is for uppercase letters, type 'a' is for lowercase letters, type 'I' for uppercase roman numbers and and 'i' for lowercase roman numbers.
Forms <form>...</form> Defines a form that is used to collect user input. An HTML form contains form elements. Form elements are different types of input elements, like text fields, checkboxes, radio buttons, submit buttons, and more.
