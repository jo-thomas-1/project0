Web programming refers to the writing, markup and coding involved in Web development, which includes Web content, Web client and server scripting and network security. The most common languages used for Web programming are HTML, CSS, JavaScript, PHP and Perl. Web programming is different from just programming, which requires interdisciplinary knowledge on the application area, client and server scripting, and database technology.

Here we focus on three tools used in web programming.

  • Hypertext Markup Language (HTML)
  • Cascading Style Sheets (CSS)
  • Syntactically Awesome Style Sheets (Sass)
